# Blockchain
Simple blockchain project for network programming course

## Run with docker
Clone repository
```console
https://github.com/zadorotskas/Blockchain.git
cd Blockchain
```
Build docker image
```console
docker build -t zadorotskas/blockchain .
```
Run docker
```console
docker-compose up
```
